== Description ==

Restaurant link offers the possibility of creating a dynamic reservation form to better manage your restaurant.

1. Create a reservation form.
1. Notify the manager by email at each booking

== Installation ==

1. Upload the `restaurant-link/` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup your settings under `Restaurant link`
1. ... Enjoy!
